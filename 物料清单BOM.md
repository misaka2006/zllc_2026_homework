## 📋 质量检验标准

### 关键元件检查
```
✓ STM32芯片: 检查Pin1标记，无划痕
✓ LM2596: 通电测试输出5V
✓ RT9013: 万用表测量引脚通断

✓ 电机: 手动转动顺滑，无卡顿
```
